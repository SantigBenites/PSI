Para correr o programa
	Abrir dois terminais
	Um no Diretorio Back-End
		Neste diretorio correr os comandos
		npm install
		npm run devstart
		O Back end deve agora estar aberto
	Um no Diretorio Front-End
		Neste diretorio correr os comandos
		npm install
		ng serve --open
		O Front end deve agora estar aberto
		
Apos estes comandos o servidor node.js deve estar a correr 
no endereco http://localhost:3000 e o servidor angular
no endereco http://localhost:4200.

Caso algum erro no processo de npm install e possivel ser 
util o programa npm audit -fix
